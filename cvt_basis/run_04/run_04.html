RUN_04
Energy Plot
for BURGERS Data

This run works with the 40 scalar flow solutions from the BURGERS dataset. It computes clusterings using 1 through 16 cluster centers, determines the corresponding clustering energy, and writes out two files that can be used as input to GNUPLOT to make plots.

  • run_04.inp, the input file.
  • run_04.out, the output of the run.
  • raw.txt, GNUPLOT data file 1.
  • raw_plot.inp, GNUPLOT commands to plot data file 1.
  • raw.ps, PostScript plot of data file 1, Energy versus Cluster Size.
  • raw2.txt, GNUPLOT data file 2.
  • raw2_plot.inp, GNUPLOT commands to plot data file 2.
  • raw2.ps, PostScript plot of data file 2, Energy versus 1/Cluster Size.
