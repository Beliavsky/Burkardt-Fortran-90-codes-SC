LEVENSHTEIN
The Levenshtein Distance Between Two Strings

LEVENSHTEIN is a FORTRAN90 library which computes the Levenshtein distance between two strings.

Languages:

LEVENSHTEIN is available in a C version and a C++ version and a FORTRAN90 version and a MATLAB version and a Python version.

Related Data and Programs:

ATBASH, a FORTRAN90 library which applies the Atbash substitution cipher to a string of text.

CAESAR, a FORTRAN90 library which can apply a Caesar Shift Cipher to a string of text.

CHRPAK, a FORTRAN90 library which works with characters and strings.

MONOALPHABETIC, a FORTRAN90 library which can apply a monoalphabetic substitution cipher to a string of text.

ROT13, a FORTRAN90 library which can encipher a string using the ROT13 cipher for letters, and the ROT5 cipher for digits.

List of Routines:

  • LEVENSHTEIN_DISTANCE computes the Levenshtein distance between strings.
  • LEVENSHTEIN_DISTANCE_TEST tests LEVENSHTEIN_DISTANCE.
  • LEVENSHTEIN_MATRIX computes the Levenshtein distance matrix between strings.
  • LEVENSHTEIN_MATRIX_TEST tests LEVENSHTEIN_MATRIX.
