PENTOMINOES
Some Pentomino Utilities

PENTOMINOES is a FORTRAN90 library which provides some utilities for manipulating pentominoes.

A pentomino is a shape formed by connecting 5 unit squares edgewise. Ignoring reflection and rotation, there are 12 distince pentominoes, nicknamed F, I, L, N, P, T, U, V, W, X, Y and Z.

List of Routines:

  • PENTOMINO_MATRIX returns a 0/1 matrix defining a particular pentomino.
  • PENTOMINO_PLOT plots a particular pentomino in a 5x5 grid.
  • TIMESTAMP prints the current YMDHMS date as a time stamp.
